This site can’t be reached #

Chrome 在訪問某些 port 時，會出現 This site can’t be reached (ERR_UNSAFE_PORT) 的錯誤。

因為 Chrome 內部有個黑名單，列出哪些 port 是不能存取的。通常自訂 port 會選 1024+，可能踩到的有 2049、3659、6000、6665-6669。
